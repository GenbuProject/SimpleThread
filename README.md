# Simple Thread

## 1. Simple Thread とは？
> "Be Simple, Be Free" をモットーに掲げる、SNSサービス及びフレームワークの総称です。
> 
> 詳しい動作につきましては[こちら](https://genbuproject.github.io/SimpleThread/)をご覧下さい。
